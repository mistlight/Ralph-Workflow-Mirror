Installation

Get up and running in minutes

Install with Cargo

The fastest way to get Ralph on your system.

cargo install ralph-cli
Requirements: Rust toolchain installed

Build from Source

Clone the repository and build locally.

git clone https://codeberg.org/mistlight/ralph.git
cd ralph
cargo install --path .

Configure Your AI Agent

Ralph works with Claude Code, Codex, or OpenCode. Set your preferred agent in ~/.ralph/config.toml:

# Ralph Configuration File

# Primary agent to use
agent = "claude-code"

# Available agents: claude-code, codex, opencode
# Fallback chain if primary fails
fallback = ["codex", "opencode"]

Features

Everything you need to automate your workflow

Preset Modes

-Q -U -S -T -L

Choose from Quick, Rapid, Standard, Thorough, or Long modes. Each preset controls how many reviewer passes Ralph performs before finalizing changes.

$ ralph -S

Custom Iterations

-D -R

Fine-tune the process with custom developer and reviewer pass counts. Override presets when you need exact control.

$ ralph -D 5 -R 20

Agent Selection

Use Claude Code, Codex, or OpenCode as your primary agent. Configure fallback chains so Ralph never gets stuck.

$ ralph --agent codex

Verbosity Control

-q -f --diagnose

Control output verbosity. Quiet mode for CI/CD, full mode for debugging, diagnose mode for deep inspection.

$ ralph -q --diagnose

CCS Integration

Claude Code Switch integration for seamless agent switching. Switch between Claude versions on the fly.

$ ralph --ccs opus

Git Integration

Automatic commit generation with descriptive messages. Ralph tracks what changed and creates meaningful commit messages.

$ ralph --no-commit

New to CLI?

Welcome to the command line! Ralph is a great tool to learn with. Here's what you need to know.

  • Terminal: The text-based interface where you run commands
  • Flag: Options that modify how a command runs (like -S)
  • Commit: Saving changes to your project's history
  • Agent: The AI model that does the actual coding work
  • Workflow: The sequence of steps from idea to completed code
Troubleshooting Common Issues

"Command not found": Make sure Ralph is installed and in your PATH. Try cargo install ralph-cli again.

"Permission denied": You may need to run with appropriate permissions. Check your Rust installation.

Agent errors: Verify your API keys are configured in ~/.ralph/config.toml.
