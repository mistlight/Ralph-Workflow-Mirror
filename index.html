03

Get started in minutes

Install Ralph and run your first automated workflow

Requirements

  • Rust toolchain (for cargo install)
  • AI agent API (Claude Code, Codex, or OpenCode)
cargo install ralph-cli

Configure your agent

Edit ~/.ralph/config.toml:

agent = "claude-code"
fallback = ["codex", "opencode"]

Your first workflow

1

Create PROMPT.md

Describe what you want to build

# Add OAuth login

Implement GitHub OAuth2 using the 'auth' crate.
Store sessions in Redis.
2

Run Ralph

Execute with your chosen preset

ralph -S
3

Review and ship

Ralph creates a commit. Review when ready.

Features

Everything you need to automate your workflow

Preset Modes

Quick, Rapid, Standard, Thorough, or Long. Choose the right iteration depth for your task.

Custom Iterations

Fine-tune with -D and -R flags for exact control over passes.

Agent Selection

Use Claude Code, Codex, or OpenCode. Configure fallback chains for reliability.

Git Integration

Automatic commits with descriptive messages. Tracks changes and creates meaningful history.

Verbosity Control

Quiet mode for CI, full mode for debugging, diagnose for deep inspection.

CCS Integration

Claude Code Switch support for seamless agent switching between versions.

New to CLI?

Welcome! Ralph is a great tool to learn with. The terminal is where you run commands—Ralph handles the complexity.

  • Terminal: The text-based interface where you type commands (like an advanced command prompt)
  • Flag: Options that modify how a command works (like -S for "Standard" mode)
  • Commit: Saving your changes to the project's history (like a save point in a game)
  • Agent: The AI assistant that writes and reviews code for you

💡 Getting started: Just install Ralph, create a PROMPT.md file describing what you want, and run ralph -S. That's it!
