What is Ralph?

Ralph is an open-source command-line tool that automates your AI coding workflow. Instead of copying prompts to a chat and pasting code back, Ralph orchestrates two specialized AI agents that collaborate to ship your changes.

You write

Create PROMPT.md describing what you want

Developer Agent

Code

Writes the implementation

Reviewer Agent

Review

Checks quality & requests fixes

Commit

Git commit created automatically

01

No copy-paste

Ralph runs locally and modifies files directly. No clipboard shuffling.

02

Self-correcting

The reviewer catches bugs and the developer fixes them—iterating until approved.

03

Yours forever

Open source (AGPL). No vendor lock-in. Runs on your machine with your API keys.

How Ralph works

Three steps from idea to committed code

01

Write your requirements

Create a PROMPT.md file in your project directory. Describe what you want to build, fix, or improve in plain English.

markdown 
# Add OAuth login

Implement GitHub OAuth2 authentication.
Use the 'oauth2' crate and store sessions in Redis.
02

Run Ralph

Execute Ralph with your preferred preset. The tool coordinates Developer and Reviewer agents to build and validate your changes.

bash 
ralph -S
Developer writes code
→
Reviewer validates
↻
∞ Iterate if needed

Review and ship

Ralph creates a meaningful git commit with your changes. Review the diff, make any adjustments, and push when ready.

Get started in minutes

Install Ralph and run your first automated workflow

Requirements

  • Rust toolchain (for cargo install)
  • AI agent API (Claude, Codex, or OpenCode)

Quick Start Steps

  1. 1Install Ralph via cargo
  2. 2Configure your AI agent
  3. 3Create PROMPT.md
  4. 4Run ralph -S
bash 
cargo install ralph-cli

Requires Rust 1.70+ and cargo in PATH

bash 
git clone https://codeberg.org/mistlight/ralph.git
cd ralph
cargo install --path .
toml 
# ~/.ralph/config.toml

agent = "claude-code"
fallback = ["codex", "opencode"]

[reviewer]
model = "claude-sonnet-4"
max_iterations = 10

Create this file at ~/.ralph/config.toml

Try it now

Terminal
$ ralph -S

→ Reading PROMPT.md

✓ Reviewer agent: Code approved

Features

Built for developers who value autonomy and quality

Multi-Agent Orchestration

Developer and Reviewer agents collaborate in real-time. The Developer writes code while the Reviewer validates quality. They iterate automatically until approval—no manual copy-paste required.

  • ✓Automatic iteration loops
  • ✓Self-correcting code quality
  • ✓Consistent review standards

Preset Iteration Modes

Five modes from Quick (3 passes) to Long (30 passes). Choose the depth for your task complexity.

Git Integration

Automatic commits with semantic messages. Full history tracking with meaningful descriptions.

Flexible Agent Selection

Claude Code, Codex, or OpenCode. Configure fallback chains for reliability.

Verbosity Control

Quiet mode for CI, full output for debugging, diagnose mode for deep inspection.

Custom Flags

Fine-tune with -D (developer) and -R (reviewer) for exact pass control.

CCS Integration

Claude Code Switch support for seamless agent version switching.
